#### Windows与Linux的区别

|          | Windows  | Linux           |
| -------- | -------- | --------------- |
| 界面     | 统一风格 | 多种桌面系统    |
| 驱动     | 硬件厂商 | 志愿者+硬件厂商 |
| 风格     | 人性化   | 专业化          |
| 用户     | 办公娱乐 | 网络/嵌入式设备 |
| 安全性   | 一般     | 较强            |
| 盈利模式 | 系统收费 | 服务收费        |

#### UNIX、Minix、GNU、Linux的历史渊源

- UNIX(贝尔实验室):最早的多用户、多任务操作系统，Unix在1979年后商业收费。
- Minix(谭宁邦):定位教学用途,没有发展壮大。
- GNU(史托曼):建立free UNIX，
  - 曲线救国：gcc/glibc/bash shell/emacs
  - Hurd诞生,为时已晚

- Linux：学习Minix，借助GNU软件。
