# 如何学习Linux开发

## 学习群体的定位

- 进阶用户:已经有大量mcu工作经验的开发者，希望晋升到薪水待遇更高

发展前景更好的linux开发中去。

- 学生用户:刚接触嵌入式Linux开发的大学生群体、电子爱好者

### 最大程度降低学习门槛

简化许多开发环境的搭建

优先学习嵌入式Linux的高频技能

鼓励用户开发产品

根据个人兴趣和工作需求，深入研究

## 传统学习路线

课程安排不合理

系统移植和驱动开发难度大，直接劝退学员

必要的基础知识不扎实(makefile\GNU C\基础数据结构\面向对象思想)
