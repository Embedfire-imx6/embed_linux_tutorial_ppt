### VSCode编辑器

#### 官网

[https://code.visualstudio.com](https://code.visualstudio.com/)

#### 安装

```
sudo dpkg -i xxx.deb
```

#### 启动

```
code
```

#### 使用特点

- 无需新建项目，直接打开文件夹
- 不同目录下的文件夹，可保存在工作区
- 默认预览模式

#### 推荐插件

- c/c++：函数跳转、自动补全

- Chinese：中文

- material theme UI

- CodeSpell Checker：拼写检查

- Bracket Pair Colorizer：括号着色器

- rainbow-highlighter：变量彩虹高光
  - shift + alt + z
  - shift + alt + a

#### 高效工作

- 移动光标到行首：home
- 移动光标到行尾：end

- 跳转文档头部：ctrl+home
- 跳转文档尾部：ctrl+end

- 花括号跳转：ctrl+shift+\
- 文件跳转：ctrl+p
- 行跳转：ctrl+g
- 批量修改：F2
