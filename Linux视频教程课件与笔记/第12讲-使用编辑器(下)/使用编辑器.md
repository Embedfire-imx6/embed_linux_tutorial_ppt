#### vi/vim编辑器

为什么要学习vi/vim编辑器

vi与vim的区别:

- vim是vi的升级版本，兼容vi
- vi按 u只能撤消上次命令，而在vim里可以无限制的撤消
- vi只能运行于类unix系统中，而vim不仅可以运行于unix,windows ,mac等多操作平台
- vim可以用不同的颜色来高亮代码
- 可通过vimrc文件配置更加高级的功能

#### vi/vim使用

进入/退出插入模式快捷键

| 快捷键 | 功能描述                                            |
| ------ | --------------------------------------------------- |
| i      | 在当前光标所在位置插入文本                          |
| a      | 在当前光标所在位置的下一个字符插入文本              |
| o      | 在光标所在位置后插入新行                            |
| r      | 替换当前光标所在位置的字符                          |
| R      | 可以替换当前光标所在位置之后的字符，按下”Esc”则退出 |
| ESC    | 退出插入模式                                        |

命令行模式快捷键

| 快捷键       | 功能描述                                           |
| ------------ | -------------------------------------------------- |
| w            | 保存文档                                           |
| w <filename> | 另存为以<filename>为文件名的文档                   |

| ! command    | 回到终端窗口，执行command命令，按回车键可切回vim。 |
